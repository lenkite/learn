Block Formatting Context

A new BFC is created any time an element
  • is the root element
  • is floated
  • has overflow: !(visible or clip)
  • has position: !(absolute or fixed)
  • has contain: layout, content or paint.
  • has display: inline-block
  • has display: inline-table or table-cell or table-caption
  • is an anonymous table cell implicitly created by the elements display: table, table-row, table-row-group, table-header-group, table-footer-group which is the default for HTML tables, table rows, table bodies, table headers, and table footers, respectively
  • is a flex item (direct child of the element with display: flex or inline-flex and is not neither a flex nor grid nor table container by itself
  • is a grid item (direct child of the element with display: grid or inline-grid and is not neither a flex nor grid nor table container by itself
  • is a Multicol container (an elements where `column-count` or `column-width` isn't auto.
